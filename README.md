# Box Office Project
Box Office is a Website, where one can view movies and book online tickets for the movies.
Languages that I've used to develop this Project are:
###### Front-End Languages:
1. HTML
2. CSS
3. BootStrap
###### Back-End Languages:
1. PHP
2. JavaScript
3. jquery
###### DataBase Technologies:
1. MySQL
